Installation Guide

  • Yssbe
  1. Install Yssbe 1.15 Beta 2 or Official Release
  2. Open Naza folder
  3. Open folder named "archive" and copy the folder that matches your Yssbe version
  4. Yssbe Shaders Download Link

  • Osbes
  1. Install latest Osbes from it's GitHub repository
  2. Make sure you're activating Vanillaccurate 64× too below Naza with the purpose of fulfilling the atlas
  3. OSBES Download Link

  • Nori
  1. Install Nori shaders
  2. Change Nori shader's subpack to 64×64 resolution and run the game
  3. Nori Shaders Download Link
